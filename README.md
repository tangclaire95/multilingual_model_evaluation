# multilingual_model_evaluation 
# Multilingual Model Advanced Evaluation

This repository contains a Python prototype for evaluating multilingual translation quality using a combination of:

- machine translation (EN → target language),
- back-translation (target language → EN),
- LLM-as-a-judge scoring (MQM-style), and
- rule-based QA checks.

The script generates a small synthetic dataset and evaluates translations into:

- Spanish (`es`)
- German (`de`)
- Chinese (`zh`)
